## 요구사항 상세

### FUN-001: 분석 진행 상태 실시간 표시

**설명**: 사용자가 분석 요청 후 현재 진행 단계를 실시간으로 확인할 수 있어야 함

**상세 내용**:
- 단계별 체크마크 표시 (질문 분석 → 문서 검색 → 관계 추적 → LLM 추론 → 추천 생성)
- 현재 단계 하이라이트
- 경과 시간 표시
- 예상 남은 시간 표시 (선택)

**수용 기준**:
- [ ] 5개 단계 각각에 상태 표시됨
- [ ] 현재 진행 중인 단계가 시각적으로 구분됨
- [ ] 경과 시간이 1초 단위로 업데이트됨

**관련 문서**: [디자인 명세 - 로딩 상태]

---

### FUN-002: 분석 작업 취소 기능

**설명**: 사용자가 진행 중인 분석을 취소할 수 있어야 함

**상세 내용**:
- "취소" 버튼 제공
- 취소 시 서버 측 작업도 중단
- 취소 후 이전 상태로 복귀

**수용 기준**:
- [ ] 취소 버튼이 로딩 중에만 표시됨
- [ ] 취소 클릭 시 2초 이내 중단
- [ ] 취소 후 폼이 다시 사용 가능 상태

**기술 참고**: AbortController 사용

---

### FUN-005: risk_review 리스크 매트릭스

**설명**: 리스크 분석 결과를 심각도/가능성 매트릭스로 시각화

**상세 내용**:
```
         가능성
         높음  중간  낮음
심각도
 높음    🔴    🟠    🟡
 중간    🟠    🟡    🟢
 낮음    🟡    🟢    🟢
```

**수용 기준**:
- [ ] 리스크가 매트릭스 셀에 매핑됨
- [ ] 셀 클릭 시 해당 리스크 상세 표시
- [ ] 색상으로 심각도 직관적 파악 가능
