## 요구사항 목록

### DES-001: 로딩 상태 시각적 피드백
**우선순위**: P0 | **상태**: 계획

**설명**:
분석 진행 중 명확하고 안심을 주는 시각적 피드백 제공

**현재 문제**:
- 단순 스피너 + 경과시간만 표시
- 무슨 작업 중인지 알 수 없음
- 오래 걸리면 "멈춘 건가?" 불안

**개선 디자인**:
```
┌─────────────────────────────────────────────┐
│ 🤖 AI가 분석 중입니다                        │
├─────────────────────────────────────────────┤
│                                             │
│  ✅ 질문 분석                    완료        │
│  ✅ 관련 문서 검색 (15개)        완료        │
│  ✅ 연관 지식 확장 (+8개)        완료        │
│  🔄 AI 추론 중...               12초        │
│  ○  추천 정보 생성              대기        │
│                                             │
│  ┌────────────────────────────────┐         │
│  │████████████░░░░░░░░░░░░░░░░░░░│ 60%     │
│  └────────────────────────────────┘         │
│                                             │
│  💡 복잡한 질문일수록 시간이 걸립니다        │
│                                             │
│           [ 취소하기 ]                       │
└─────────────────────────────────────────────┘
```

**디자인 가이드라인**:
- 단계별 상태: 아이콘 + 텍스트 + 시간/상태
- 진행률 바: 전체 예상 대비 현재 위치
- 취소 버튼: 눈에 띄지만 실수 방지
- 팁 메시지: 긍정적, 안심 유도

### DES-002: 모드별 아이콘 및 색상 체계
**우선순위**: P0 | **상태**: 계획

**설명**:
각 분석 모드를 직관적으로 구분하는 아이콘과 색상 정의

**모드별 비주얼 가이드**:

| 모드 | 아이콘 | 색상 | 의미 |
|------|--------|------|------|
| design_explain | 📐 또는 설계도 아이콘 | 보라 (#8b5cf6) | 구조, 설계 |
| risk_review | ⚠️ 또는 방패 아이콘 | 주황 (#f59e0b) | 주의, 경고 |
| next_steps | 🚀 또는 화살표 아이콘 | 녹색 (#16a34a) | 진행, 성장 |
| history_trace | 📜 또는 시계 아이콘 | 인디고 (#6366f1) | 과거, 기록 |

**적용 위치**:
- 모드 선택 드롭다운
- 결과 헤더
- 탭/섹션 구분
- 내보내기 문서

### DES-003: 결과 영역 정보 계층 구조
**우선순위**: P1 | **상태**: 계획

**설명**:
분석 결과를 스캔하기 쉬운 계층 구조로 구성

**정보 계층**:
```
Level 1: 핵심 요약 (3줄)
    ↓
Level 2: 상세 분석 (접힘 가능)
    ↓
Level 3: 참조 정보 (탭)
    ↓
Level 4: 추천/탐색 (사이드 또는 하단)
```

**시각적 구분**:
```
┌─────────────────────────────────────────────┐
│ [모드 아이콘] 설계 분석 결과                  │  ← 모드 색상 강조
├─────────────────────────────────────────────┤
│                                             │
│ 📊 핵심 요약                                 │  ← Level 1 (항상 표시)
│ ─────────────────────────────────────────── │
│ • 포인트 1                                  │
│ • 포인트 2                                  │
│ • 포인트 3                                  │
│                                             │
│ ▾ 📋 상세 분석                              │  ← Level 2 (접힘 가능)
│ ┌───────────────────────────────────────┐   │
│ │ 상세 내용...                          │   │
│ └───────────────────────────────────────┘   │
│                                             │
│ [청크] [문서] [관계]                         │  ← Level 3 (탭)
│ ┌───────────────────────────────────────┐   │
│ │ 탭 내용...                            │   │
│ └───────────────────────────────────────┘   │
│                                             │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│ 🔗 관련 정보        [접기/펼치기]            │  ← Level 4
├─────────────────────────────────────────────┤
│ • 관련 청크 카드들...                        │
│ • 추천 라벨 태그들...                        │
└─────────────────────────────────────────────┘
```

### DES-004: 시각화 요소 반응형 디자인
**우선순위**: P1 | **상태**: 계획

**설명**:
차트, 다이어그램 등 시각화 요소가 화면 크기에 맞게 조정

**브레이크포인트**:
```css
/* Desktop */
@media (min-width: 1024px) {
  .visualization { width: 100%; max-width: 800px; }
}

/* Tablet */
@media (min-width: 768px) and (max-width: 1023px) {
  .visualization { width: 100%; }
}

/* Mobile */
@media (max-width: 767px) {
  .visualization {
    width: 100%;
    overflow-x: auto; /* 가로 스크롤 허용 */
  }
}
```

**시각화별 반응형 전략**:
| 시각화 | Desktop | Tablet | Mobile |
|--------|---------|--------|--------|
| 다이어그램 | 전체 표시 | 축소 | 가로 스크롤 |
| 리스크 매트릭스 | 5x5 | 5x5 | 3x3 + 상세 |
| 타임라인 | 가로 | 가로 | 세로 |
| 간트 차트 | 전체 | 축소 | 목록 형태 |

### DES-005: 다크 모드 지원
**우선순위**: P2 | **상태**: 계획

**설명**:
시스템 설정 연동 및 수동 전환 가능한 다크 모드

**다크 모드 색상**:
```css
[data-theme="dark"] {
  --bg-primary: #1f2937;
  --bg-secondary: #374151;
  --text-primary: #f9fafb;
  --text-secondary: #d1d5db;
  --border: #4b5563;

  /* 모드 색상 (밝기 조정) */
  --mode-design: #a78bfa;
  --mode-risk: #fbbf24;
  --mode-next: #34d399;
  --mode-history: #818cf8;
}
```

**전환 방식**:
1. 시스템 설정 자동 감지 (`prefers-color-scheme`)
2. 헤더에 토글 버튼 (☀️/🌙)
3. 설정 저장 (localStorage)

### DES-006: 접근성 가이드라인 준수
**우선순위**: P2 | **상태**: 계획

**설명**:
WCAG 2.1 AA 수준 접근성 준수

**체크리스트**:
- [ ] 색상 대비: 4.5:1 이상 (텍스트), 3:1 이상 (UI)
- [ ] 키보드 내비게이션: Tab/Enter/Escape
- [ ] 스크린 리더: ARIA 레이블
- [ ] 포커스 표시: 명확한 아웃라인
- [ ] 애니메이션: `prefers-reduced-motion` 존중

**ARIA 예시**:
```html
<div role="progressbar"
     aria-valuenow="60"
     aria-valuemin="0"
     aria-valuemax="100"
     aria-label="분석 진행률 60%">
</div>

<button aria-label="분석 취소">
  취소하기
</button>

<div role="alert" aria-live="polite">
  분석이 완료되었습니다.
</div>
```

## UI 컴포넌트 명세

### 로딩 스테퍼 (Loading Stepper)
```
Props:
- steps: Array<{ id, label, status: 'pending'|'active'|'done' }>
- currentStep: number
- elapsedTime: number

States:
- pending: 회색 원, 회색 텍스트
- active: 파란색 원 + 스피너, 검정 텍스트, 시간 표시
- done: 녹색 체크, 회색 텍스트
```

### 모드 선택기 (Mode Selector)
```
Props:
- modes: Array<{ id, icon, label, description }>
- selected: string
- onChange: (modeId) => void

Hover 시:
- 배경 밝아짐
- 툴팁으로 상세 설명
```

### 결과 카드 (Result Card)
```
Props:
- title: string
- icon: string
- color: string (모드 색상)
- summary: string[]
- children: ReactNode (상세 내용)
- collapsible: boolean

States:
- expanded / collapsed
```
