# docs/devtest/reports — Task별 실행 결과 리포트

Phase 11 통합 테스트 **Task당 실행 결과 리포트**를 저장하는 디렉토리입니다.

---

## 파일 명명

- `phase11-4-2-task-11-X-Y-execution-report.md`
- 예: `phase11-4-2-task-11-1-1-execution-report.md`

---

## 포함 항목

- **코드 오류**: 재현 조건, 로그/메시지, 스택(요약)
- **미해결 이슈**: 설명, 우선순위, 영향 범위
- **해결된 이슈**: 해결 방법, 커밋/PR 참조
- **시나리오 결과 요약**: 시나리오 ID별 통과/실패
