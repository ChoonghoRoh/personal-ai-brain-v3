## 1. 개요

### 1.1 목표

Reasoning 결과에 **공통 결과 구조**(요약 / 상세 / 시각화 / 인사이트)를 적용하여 모드·표시 방식이 달라도 일관된 레이아웃과 데이터 형식을 갖추게 한다.

### 1.2 구조 정의

| 영역     | 내용                                          |
| -------- | --------------------------------------------- |
| 요약     | 한 줄 또는 짧은 요약 텍스트                   |
| 상세     | 본문·설명 텍스트                              |
| 시각화   | Mermaid/Chart/로드맵/타임라인 등 (Phase 10-2) |
| 인사이트 | 핵심 인사이트·추천·다음 액션                  |
