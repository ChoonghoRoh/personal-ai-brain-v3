## 1. 개요

### 1.1 목표

Reasoning 페이지 및 시각화 영역에 **다크 모드**를 지원하여 사용자 선호·환경에 맞는 테마를 제공한다.

### 1.2 대상

| 대상             | 내용                                                    |
| ---------------- | ------------------------------------------------------- |
| CSS/테마         | 다크 모드 색상·배경·텍스트                              |
| 토글/연동        | 사용자 토글 또는 시스템 설정(prefers-color-scheme) 연동 |
| Reasoning·시각화 | 결과 영역·차트 등 다크 테마 적용                        |

---

## 2. 파일 변경 계획

### 2.1 수정

| 파일 경로                        | 용도                              |
| -------------------------------- | --------------------------------- |
| CSS/테마 파일                    | 다크 모드 변수·클래스 정의        |
| Reasoning 페이지·시각화 컴포넌트 | 다크 모드 클래스 적용             |
| (선택) 설정 저장                 | 사용자 선택 저장(localStorage 등) |

- [ ] 시각화 영역(Mermaid, Chart 등) 다크 모드 적용
- [ ] 대비·가독성 확인
