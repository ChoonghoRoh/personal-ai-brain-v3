## ✅ 수정 내용

### 1. 파일 수정
- `web/public/js/ask.js`: `askQuestion` 함수를 6개의 작은 함수로 분리

### 2. 분리된 함수
1. **`prepareQuestionRequest()`**: 질문 요청 데이터 준비
2. **`initializeAskUI()`**: UI 초기화
3. **`executeQuestion(requestData)`**: 질문 API 호출
4. **`displayAnswer(data)`**: 답변 표시
5. **`displaySources(sources)`**: 참고 문서 표시
6. **`updateConversationHistory(question, data)`**: 대화 기록 업데이트
7. **`restoreAskUI()`**: UI 상태 복원
8. **`askQuestion()`**: 메인 함수 (리팩토링 후)
