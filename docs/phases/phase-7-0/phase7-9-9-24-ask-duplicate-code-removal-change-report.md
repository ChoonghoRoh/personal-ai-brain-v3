## 📋 변경 개요

Ask.js의 중복 코드를 공통 함수로 추출하여 코드 재사용성을 향상시켰습니다.

---

## 🔧 변경된 파일

### 1. `web/public/js/ask.js`
- **변경 유형**: 리팩토링 (중복 코드 제거)
- **주요 변경사항**:
  - 중복된 로딩/에러 메시지 표시 패턴을 공통 함수로 추출
  - 2개의 공통 함수 추가

---

## 📝 상세 변경 내용

### 변경: 공통 함수 추출

**추출된 공통 함수**:
1. **`showLoading(elementId, message)`**: 로딩 메시지 표시
2. **`showError(elementId, message)`**: 에러 메시지 표시

**수정된 함수**:
- `initializeAskUI()`: 로딩 메시지 표시 → `showLoading()` 사용
- `askQuestion()`: 에러 메시지 표시 → `showError()` 사용
