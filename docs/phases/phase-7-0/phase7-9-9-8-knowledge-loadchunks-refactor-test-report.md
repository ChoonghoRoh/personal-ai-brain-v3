Knowledge.js의 `loadChunks` 함수(140줄)를 여러 작은 함수로 분리하여 코드 가독성과 유지보수성을 향상시켰습니다.

## ✅ 수정 내용

### 1. 파일 수정
- `web/public/js/knowledge.js`: `loadChunks` 함수를 6개의 작은 함수로 분리

### 2. 분리된 함수
1. **`buildChunksUrl()`**: 청크 API URL 구성
2. **`fetchChunks()`**: 청크 데이터 가져오기
3. **`renderEmptyState()`**: 빈 상태 표시
4. **`renderChunkCard(chunk)`**: 청크 카드 HTML 생성
5. **`renderChunkList(chunks)`**: 청크 목록 렌더링
6. **`setupChunkEventListeners()`**: 청크 카드 이벤트 리스너 설정
7. **`renderErrorState(error)`**: 에러 상태 표시
8. **`loadChunks()`**: 메인 함수 (리팩토링 후)
