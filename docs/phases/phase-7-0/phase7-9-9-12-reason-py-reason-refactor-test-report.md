3. **`expand_chunks_with_relations(...)`**: 관계를 통해 청크 확장
4. **`add_semantic_search_results(...)`**: Qdrant 의미 검색 결과 추가
5. **`build_context_chunks(...)`**: 컨텍스트 청크 구성
6. **`collect_relations(...)`**: 관계 정보 수집
7. **`generate_reasoning_answer(...)`**: Reasoning 답변 생성
8. **`reason(...)`**: 메인 함수 (리팩토링 후)
