## 📝 상세 변경 내용

### 변경: 주석 추가

**추가된 주석**:
1. **`displayHistory()`**: 검색 히스토리 표시 함수에 JSDoc 주석 추가
2. **`searchFromHistory(query)`**: 검색 히스토리에서 검색 실행 함수에 JSDoc 주석 추가
3. **`highlightText(text, query)`**: 텍스트 하이라이팅 함수에 JSDoc 주석 추가
4. **`loadRecommended()`**: 추천 문서 로드 함수에 JSDoc 주석 추가
5. **`search()`**: 검색 실행 함수에 JSDoc 주석 추가
