## ✅ 수정 내용

### 1. 파일 수정
- `web/public/js/dashboard.js`: 중복된 로딩/에러/빈 상태 메시지 표시 패턴을 공통 함수로 추출

### 2. 추출된 공통 함수
1. **`showLoading(elementId, message)`**: 로딩 메시지 표시
2. **`showEmptyState(elementId, message)`**: 빈 상태 메시지 표시
3. **`showError(elementId, message)`**: 에러 메시지 표시

### 3. 수정된 부분
- `renderRecentWork()`: 빈 상태 메시지 표시 → `showEmptyState()` 사용
- `renderRecentDocuments()`: 빈 상태 메시지 표시 → `showEmptyState()` 사용
- `loadDashboard()`: 에러 메시지 표시 → `showError()` 사용
- `renderActivityChart()`: 빈 상태 메시지 표시 → `showEmptyState()` 사용
- `loadDocuments()`: 에러 메시지 표시 → `showError()` 사용
- `displayDocuments()`: 빈 상태 메시지 표시 → `showEmptyState()` 사용
