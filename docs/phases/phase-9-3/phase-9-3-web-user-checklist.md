## 기능 Flow 요약 (브라우저)

| Flow                        | 경로 순서                                                                  | 확인 항목                  |
| --------------------------- | -------------------------------------------------------------------------- | -------------------------- |
| **검색 → 결과**             | /search → 검색 실행 → 결과 클릭                                            | 모드별 결과, 상세 이동     |
| **질의 → 답변**             | /ask → 질문 입력 → 전송 → 답변·관련 문서                                   | 옵션별 동작                |
| **추론 → 관련 정보**        | /reason → 질문·실행 → 관련 정보 토글·샘플 질문 클릭 → 재추론               | recommendations, UI        |
| **지식 → 상세 → 라벨/관계** | /knowledge → 항목 클릭 → /knowledge-detail → 라벨 매칭 or 관계 매칭 → 적용 | 추천·적용 반영             |
| **승인 → 관계 적용**        | /knowledge-admin 또는 /admin/approval → 청크 승인 → 관계 추천 → 적용       | suggested_relations, apply |
| **청크 관리**               | /admin/chunk-labels → 검색·필터 → 청크 선택 → 라벨 추가/제거               | 페이징, 추천 라벨          |
| **청크 생성**               | /admin/chunk-create → 파일 선택 → 분할/병합 → 등록 방식·경로 입력 → 등록   | 스토리지 기반, 3단계       |
