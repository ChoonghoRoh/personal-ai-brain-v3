## §1 목표

검색 입력과 깊이 필터를 트리 패널 헤더에 컴팩트하게 통합.
기존 분산 배치된 UI 요소를 하나의 영역으로 집중.

| `web/src/pages/admin/groups.html` | **수정** (HTML 구조 변경) |

## §3 구현 상세

### 3.1 HTML 구조 변경

기존 `groups-search-bar` 제거 → `kg-tree-header` 내부로 이동:

```
kg-tree-header
├── kg-tree-header-row (flex: 타이틀 + 깊이 select)
│   ├── "키워드 그룹" 타이틀
│   └── 깊이 select (3/5/전체)
└── kg-tree-search
    └── 검색 입력 (onkeyup → searchGroupsAndKeywords)
```

### 3.2 기존 요소 제거

- `groups-search-bar` HTML 및 CSS 제거
- `kg-data-level-section`(깊이 필터)을 kg-detail-panel에서 제거 → 트리 헤더로 이동

### 3.3 반응형

768px 이하: 트리 패널 폭 조정, 검색 입력 축소.

## §4 완료 기준 (Done Definition)

1. 트리 헤더에 검색 입력 + 깊이 셀렉트 컴팩트 배치
2. 기존 `groups-search-bar` 및 `kg-data-level-section` 제거
3. 검색 기능 정상 동작 (그룹명 검색 + 트리 하이라이트)
4. 깊이 필터 변경 시 트리 즉시 갱신
5. 768px 이하 반응형 레이아웃 정상
