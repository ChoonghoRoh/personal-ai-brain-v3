## §3 테스트 항목

### 3.1 트리 렌더링
- [ ] 트리 정상 로드 (API `/api/labels/tree`)
- [ ] 폴더 아이콘 표시: 📁 (접힘), 📂 (펼침), 리프 노드 아이콘 없음
- [ ] 접기/펼치기 토글 정상 동작
- [ ] Breadcrumb 정상 표시

### 3.2 노드 이동
- [ ] 우클릭 → "루트로 이동" 정상 동작
- [ ] 우클릭 → "부모 변경" → 대상 선택 → 이동 확인
- [ ] 상세 패널 부모 변경 드롭다운 → 저장 → 이동 확인
- [ ] 낙관적 UI: 이동 즉시 트리 갱신, API 실패 시 복구 확인
- [ ] 순환 참조 시 에러 메시지 표시 확인

### 3.3 CRUD
- [ ] 새 그룹 생성: 이름+설명+색상 입력 → 저장 → 트리에 반영
- [ ] 인라인 편집: 그룹 선택 즉시 편집 폼 → 수정 → 저장 → 반영
- [ ] 그룹 삭제: 영향도 표시 → 확인 → 삭제 → 트리에서 제거
- [ ] AI 키워드 추천: 설명 입력 → 추천 → 선택 → 저장 시 연결

### 3.4 검색
- [ ] 트리 헤더 검색: 그룹명 입력 → 매칭 노드 하이라이트
- [ ] 키워드 검색: 키워드명 입력 → 필터링
- [ ] 깊이 필터 변경 → 트리 즉시 갱신

### 3.5 키워드 관리
- [ ] 그룹 키워드 / 그룹 외 키워드 섹션 구분 표시
- [ ] 키워드 선택 → 연결 정상
- [ ] 키워드 선택 → 해제 정상
- [ ] 전체 선택/해제 버튼 정상
- [ ] 유사 키워드 섹션 정상

### 3.6 반응형
