## 🚀 2차 목표 개요

### 1️⃣ 자동 기록 업데이트 시스템

**목표**

- 문서 변경 시 자동 감지
- 변경된 파일만 재임베딩
- Qdrant 데이터 자동 업데이트
- 변경 이력 자동 기록

**예정 작업**

- `watcher.py` 생성
- brain 디렉토리 변경 감지
- 변경 문서만 재처리 로직 구현
- 자동 로그 업데이트

---

### 2️⃣ Git 자동 커밋 시스템

**목표**

- 시스템이 스스로 성장 기록을 남기는 구조 구축

**예정 작업**

- Python subprocess 기반 Git 연동
- 자동 add / commit
- 커밋 메시지 규칙 정의
- push는 선택형

---

### 3️⃣ 문서 자동 수집 확장

**목표**

- Markdown 중심에서 확장
- 다양한 문서 포맷 자동 처리

**예정 작업**

- `/collector/` 디렉토리 추가
- PDF 지원
- DOCX 지원
- (HWP는 변환 후 처리 방향)

---

### 4️⃣ 서브 에이전트(관리 AI) 1차 버전

**목표**

- 시스템 관리 AI 개념 도입
- 상태 점검 / 요약 / TODO 자동화

**예정 작업**

- `/brain/system/` 생성
- context 자동 유지
- status 자동 생성
- todo 자동 갱신
