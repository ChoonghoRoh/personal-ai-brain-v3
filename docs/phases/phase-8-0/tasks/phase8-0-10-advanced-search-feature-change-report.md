## 🔧 변경 사항 상세

### 1. 고급 검색 API (`backend/routers/search.py`)

#### AdvancedSearchRequest 모델

**필드**:
- `query`: 검색어
- `operator`: 연산자 (AND, OR, NOT)
- `file_types`: 파일 형식 필터
- `project_ids`: 프로젝트 필터
- `label_ids`: 라벨 필터
- `date_from`, `date_to`: 날짜 범위
- `limit`, `offset`: 페이징

#### 새로운 엔드포인트

1. **POST `/api/search/advanced`**
   - 고급 검색 실행
   - 복합 필터링 지원

2. **GET `/api/search/filters`**
   - 필터 옵션 조회
   - 파일 형식, 프로젝트 목록
