| 파일명 패턴 | 설명 |
| ----------- | ----- |
| `phase8-0-N-task.md` | Task 실행 계획 (N = todo 항목 순번, 1부터) |
| `phase8-0-N-task-test-result.md` | Task 테스트 결과 |

- Task ID 개념: `task-8-0-N`
- 상위 Phase 문서(plan, todo-list, summary, test-report, change-report)는 `../` (phase-8-0 폴더)에 위치한다.
