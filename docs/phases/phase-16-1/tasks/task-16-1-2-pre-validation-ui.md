## §1. 개요

파일 선택 체크박스 변경 시 선택 문서 수·예상 청크·예상 소요 시간을 실시간 표시하고, 50개 초과 선택 시 경고를 노출하여 대량 처리 리스크를 사전에 안내한다.

참조: [리스크 분석 §3.1 방안 C](../../../planning/260217-1600-AI자동화기능-리스크분석.md)

## §3. 작업 체크리스트 (Done Definition)

- [ ] 체크박스 선택/해제 시 요약 영역에 "선택: N개 | 예상 청크: ~M | 예상 소요: ~K분" 갱신
- [ ] 예상 청크 = selected.length * 12, 예상 소요 = ceil(예상청크/10) 분 계산
- [ ] 50개 초과 선택 시 warn 스타일 적용 + "대량 처리 — 배치 분할 권장" 메시지
- [ ] 기존 AI 자동화 페이지 동작·레이아웃에 영향 없음
- [ ] 페이지 로드 시 콘솔 에러 0건
