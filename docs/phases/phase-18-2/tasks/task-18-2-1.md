## §1 목적
Document의 file_path를 파싱하여 가상 폴더 트리 구조를 반환하는 API.

## §2 API 스펙
```
GET /api/knowledge/folder-tree?project_id={id}
```
응답:
```json
{
  "name": "root",
  "type": "folder",
  "children": [
    {
      "name": "backend",
      "type": "folder",
      "children": [
        {"name": "labels.py", "type": "file", "document_id": 1, "chunk_count": 5}
      ]
    }
  ]
}
```

## §3 구현 상세
1. project_id로 Document 목록 조회
2. file_path를 "/" 기준 split → 폴더 계층 딕셔너리 구성
3. 트리 형태로 변환하여 반환
