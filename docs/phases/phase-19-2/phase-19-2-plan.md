# Phase 19-2 Plan: 통계 메뉴 레이아웃 변경

> **Phase**: 19-2
> **우선순위**: P2
> **도메인**: FE 전용 (BE 변경 없음)
> **Master Plan**: [phase-19-master-plan.md](../phase-19-master-plan.md)

---

## 목표

통계 대시보드(`/admin/statistics`)의 레이아웃을 **반응형 CSS Grid 기반**으로 재구성하여 Desktop/Tablet/Mobile 3단계 반응형 지원.

## 현재 상태 (AS-IS)

- `statistics.css` (324줄): `grid-template-columns: repeat(auto-fit, minmax(N, 1fr))` 패턴 사용
- 반응형: `@media (max-width: 768px)` 1단계만 존재
- 4개 Tier 구조: 요약 → 분포 → 트렌드 → 상세

## TO-BE 레이아웃

```
┌─────────────────────────────────────────┐
│ Statistics Dashboard       [새로고침]    │ ← 1단 헤더
├─────────────────────────────────────────┤
│ [총 문서] [총 청크] [총 라벨] [오늘 사용] │ ← 요약 1행 4단
├─────────────────────────────────────────┤
│ [문서유형 분포] [청크상태 분포] [라벨 분포]│ ← 분포 1행 3단
├─────────────────────────────────────────┤
│ [일별 트렌드 (최근 7일) 차트]             │ ← 트렌드 1행
├─────────────────────────────────────────┤
│ [인기 라벨 TOP 10]  |  [프로젝트별 현황]  │ ← 상세 2단 좌우
└─────────────────────────────────────────┘
```

## 반응형 기준점

| 해상도 | 요약 카드 | 분포 차트 | 상세 현황 |
|--------|:--------:|:--------:|:--------:|
| Desktop (1200px+) | 4단 1행 | 3단 1행 | 2단 좌우 |
| Tablet (768~1200px) | 2단 2행 | 2단 | 2단 좌우 |
| Mobile (~768px) | 1단 | 1단 | 1단 상하 |
