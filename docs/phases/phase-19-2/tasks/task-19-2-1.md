## 목표

통계 대시보드 CSS를 명시적 CSS Grid 기반으로 재구성하여 3단계 반응형 레이아웃 구현.

## 변경 사항

### 1. 반응형 브레이크포인트 추가

현재 768px 1개 → **1200px, 768px, 480px** 3단계로 확장.

### 2. 각 섹션별 Grid 규칙

**요약 카드 (`.summary-cards`)**:
- Desktop (1200px+): `grid-template-columns: repeat(4, 1fr)` — 4단 1행
- Tablet (768~1200px): `grid-template-columns: repeat(2, 1fr)` — 2단 2행
- Mobile (~768px): `grid-template-columns: 1fr` — 1단

**분포 차트 (`.charts-row`)**:
- Desktop (1200px+): `grid-template-columns: repeat(3, 1fr)` — 3단 1행
- Tablet (768~1200px): `grid-template-columns: repeat(2, 1fr)` — 첫 2개 좌우, 3번째 full-width (`grid-column: 1 / -1` 또는 2단 유지)
- Mobile (~768px): `grid-template-columns: 1fr` — 1단

**트렌드 (`.trends-section`)**:
- 모든 해상도: full-width (변경 불필요)

**상세 현황 (`.details-section`)**:
- Desktop (1200px+): `grid-template-columns: repeat(2, 1fr)` — 2단 좌우
- Tablet (768~1200px): `grid-template-columns: repeat(2, 1fr)` — 2단 유지
- Mobile (~768px): `grid-template-columns: 1fr` — 1단 상하

### 3. auto-fit → 명시적 columns

현재 `repeat(auto-fit, minmax(N, 1fr))`를 명시적 `repeat(N, 1fr)`로 변경하여 정확한 단수 제어.

### 4. Tier 헤더 정리

`.stats-tier`, `.tier-header`, `.tier-subtitle` — 기존 스타일 유지. 불필요한 중복만 정리.

## 주의 사항

- 기존 카드 클릭 필터링(`.filtered-list-section`) CSS는 건드리지 않음
- CSS 변수 (`var(--card-bg)` 등) 유지
- `chart-container` 높이(250px/300px) 유지
- 파일 500줄 초과 사전 방지 (REFACTOR-3)

## 완료 기준

- [ ] Desktop (1200px+): 요약 4단, 분포 3단, 트렌드 full, 상세 2단
- [ ] Tablet (768~1200px): 요약 2단, 분포 2단, 상세 2단
- [ ] Mobile (~768px): 전체 1단
- [ ] 기존 스타일 호환 유지 (카드 hover, active 등)
