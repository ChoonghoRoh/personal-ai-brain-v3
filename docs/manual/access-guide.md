# 접근 가이드

## Knowledge Admin (지식 관리) 접근 방법

### 방법 1: 헤더 메뉴에서 접근

모든 페이지의 상단 헤더에 있는 네비게이션 메뉴에서 **"⚙️ 지식 관리"** 메뉴를 클릭하세요.

**경로**: 모든 페이지 상단 헤더 → ⚙️ 지식 관리

### 방법 2: 직접 URL 접근

브라우저 주소창에 다음 URL을 입력하세요:

```
http://localhost:8001/knowledge-admin
```

### 방법 3: 대시보드에서 접근

대시보드(`/dashboard`) 페이지의 **"⚙️ 지식 관리"** 섹션에서 **"⚙️ Knowledge Admin 열기"** 버튼을 클릭하세요.

## Knowledge Admin 기능

### 탭 1: 라벨 관리

- **라벨 생성**: 새 라벨을 생성합니다
- **라벨 목록**: 등록된 모든 라벨을 확인합니다
- **라벨 삭제**: 불필요한 라벨을 삭제합니다
- **청크 라벨 관리**: 특정 청크에 라벨을 부여하거나 제거합니다

### 탭 2: 청크 승인 센터

- **상태별 필터**: Draft / Approved / Rejected 상태별로 청크를 필터링합니다
- **청크 승인**: Draft 상태의 청크를 승인합니다
- **청크 거절**: 부적절한 청크를 거절합니다
- **청크 상세 보기**: 청크 내용, 라벨, AI 추천을 확인합니다
- **AI 라벨 추천**: AI가 추천한 라벨을 확인하고 적용합니다
- **AI 유사 청크 추천**: 유사한 청크를 확인합니다

## 문제 해결

### 헤더 메뉴가 표시되지 않을 때

1. 브라우저 콘솔(F12)에서 오류 메시지를 확인하세요
2. `/static/js/header-component.js` 파일이 로드되었는지 확인하세요
3. 페이지를 새로고침하세요

### 페이지가 로드되지 않을 때

1. 서버가 실행 중인지 확인하세요 (`http://localhost:8001/health`)
2. 브라우저 콘솔에서 오류 메시지를 확인하세요
3. 네트워크 탭에서 요청 상태를 확인하세요

### 직접 URL 접근

위의 방법들이 작동하지 않으면, 브라우저 주소창에 직접 입력하세요:

```
http://localhost:8001/knowledge-admin
```
