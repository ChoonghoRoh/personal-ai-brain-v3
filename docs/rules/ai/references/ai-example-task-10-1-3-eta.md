## 1. 개요

### 1.1 목표

분석 **시작 시 예상 소요 시간**을 사용자에게 안내하여 대기 기대치를 설정한다.

### 1.2 대상

| 대상      | 내용                                        |
| --------- | ------------------------------------------- |
| 추정 로직 | 경험치·휴리스틱 기반 예상 시간 (초 단위 등) |
| UI        | 분석 시작 시 예상 시간 안내 문구 표시       |

## 2. 파일 변경 계획

### 2.1 수정

| 파일 경로                                          | 용도                       |
| -------------------------------------------------- | -------------------------- |
| 프론트엔드 (Reasoning 페이지 또는 10-1-1 컴포넌트) | 예상 시간 표시 영역        |

## 3. 작업 체크리스트

### 3.1 추정 로직

- [ ] 예상 소요 시간 추정 로직 (경험치 또는 휴리스틱)
  - [ ] 모드·필터·문서 수 등 간단한 요인 반영 가능
  - [ ] 고정값(예: 약 30초~1분)으로 시작 후 점진 개선 가능

### 3.2 UI

- [ ] 분석 시작 시 UI에 예상 시간 안내 문구 표시
  - [ ] 예: "예상 소요 시간: 약 30초~1분"
  - [ ] 10-1-1 진행 상태 컴포넌트 근처 또는 로딩 영역에 배치
