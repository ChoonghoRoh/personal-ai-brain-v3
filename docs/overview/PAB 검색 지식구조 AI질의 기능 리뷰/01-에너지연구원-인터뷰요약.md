## 현재 기능 사용 경험

- **검색**: 대형 보고서 내 키워드/의미 검색을 활용해 근거 문장 추출
- **지식 구조**: 라벨/관계가 존재하지만 대량 문서에서 구조화 흐름이 분절
- **AI 질의**: 답변은 빠르나 출처 표기가 약해 내부 검토에 부담

---

## 사용 불편점

| 불편점                             | 심각도 | 영향             |
| ---------------------------------- | ------ | ---------------- |
| 검색 결과 근거(출처) 표기 부족     | 상     | 보고서 검증 지연 |
| 검색·지식 구조·AI 질의 필터 불일치 | 상     | 업무 흐름 단절   |
| 대량 문서용 일괄 처리 부족         | 중     | 반복 작업 증가   |

---

## 변경 요구사항

1. 검색 결과에 **출처·매칭 타입** 표시(키워드/벡터)
2. 라벨/프로젝트/승인 상태 **공통 필터** 제공
3. 대량 문서 **일괄 라벨/일괄 승인** 개선

---

## 업그레이드·통합 시나리오

- **옵션 A**: 검색 결과 → 지식 구조 하이라이트 → AI 질의(선택 청크 기반)
- **옵션 B**: 검색 결과 일괄 선택 → 문서 재조합/내보내기

---

## 기대 효과

- 대형 보고서 근거 추출 시간 50% 단축
- 검색/AI 답변 신뢰도 상승
