# 7. 창업랩 (송창업 대표) — 풀 리뷰

**요약본**: [07-창업랩-인터뷰요약.md](07-창업랩-인터뷰요약.md)
**모델 주석**: GPT-5.2-Codex 단일 모델 사용(모델 교체 불가)

---

## 1) 현재 기능 설명 및 사용 흐름

### 검색

- 시장/기술 키워드로 자료 탐색

### 지식 구조

- 라벨/관계로 아이디어 구조화 가능
- 버전 관리/스냅샷 기능 부족

### AI 질의

- 아이디어 정리/요약에 유용
- 외부 공유/협업 기능 제한

---

## 2) 사용 후기

- **좋은 점**: 검색 품질과 AI 요약이 빠름
- **아쉬운 점**: 피벗을 위한 버전/비교 기능 부족

---

## 3) 사용 불편점(상세)

| 구분 | 문제                | 영향                  |
| ---- | ------------------- | --------------------- |
| 버전 | Diff/버전 비교 없음 | 피벗 판단 지연        |
| 공유 | 공유 링크/권한 없음 | 멘토/투자자 공유 지연 |
| 실험 | 임시 청크 모드 없음 | 실험 속도 저하        |

---

## 4) 변경해야 할 사항(우선순위)

1. **버전 관리 + Diff 비교**
2. **공유 링크(만료/권한)**
3. **빠른 실험 모드**(임시 청크)

---

## 5) 업그레이드/변경/통합 시나리오

- **통합 시나리오 A**: 검색 결과 → 지식 구조 스냅샷 → 피벗 비교
- **통합 시나리오 B**: AI 질의 결과 공유 링크 생성

---

## 6) QA/보안/성능 관점

- 공유 링크 권한/만료 테스트
- 버전 스냅샷 무결성 검증
- 임시 청크 자동 정리 정책 확인

---

## 7) 테스트 시나리오 제안

### pytest

- 버전 스냅샷 생성/복원 테스트
- 공유 링크 권한 검증

### Playwright

- 버전 비교 화면 흐름
- AI 질의 결과 공유 링크 확인

---

## 8) 기대 효과

- 피벗 검증 속도 50% 개선
- 협업 효율 향상
