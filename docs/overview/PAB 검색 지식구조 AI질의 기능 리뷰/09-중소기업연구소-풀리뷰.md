# 9. 중소기업연구소 (조중소 이사) — 풀 리뷰

**요약본**: [09-중소기업연구소-인터뷰요약.md](09-중소기업연구소-인터뷰요약.md)
**모델 주석**: GPT-5.2-Codex 단일 모델 사용(모델 교체 불가)

---

## 1) 현재 기능 설명 및 사용 흐름

### 검색

- 기본 검색은 가능하나 가이드 부족

### 지식 구조

- 라벨/관계 설정이 복잡

### AI 질의

- 기능은 있으나 자동화/설정 부담 큼

---

## 2) 사용 후기

- **좋은 점**: 기능은 강력하나 초보자에게 복잡

---

## 3) 사용 불편점(상세)

| 구분 | 문제             | 영향             |
| ---- | ---------------- | ---------------- |
| 도입 | 온보딩 부족      | 도입 실패 위험   |
| 언어 | 완전 한글화 부족 | 현장 적용 어려움 |
| 운영 | 전자동 모드 부재 | 인력 부담 증가   |

---

## 4) 변경해야 할 사항(우선순위)

1. **초보자 모드/튜토리얼**
2. **완전 한글화**
3. **전자동 모드**(업로드→분할→승인→라벨)

---

## 5) 업그레이드/변경/통합 시나리오

- **통합 시나리오 A**: 단계별 가이드 + 자동화 플로우
- **통합 시나리오 B**: 검색/AI 결과를 즉시 보고서로 내보내기

---

## 6) QA/보안/성능 관점

- 초보자 모드 이탈률 측정
- 자동화 작업 실패 시 롤백 검증
- 한글 UI 품질 테스트

---

## 7) 테스트 시나리오 제안

### pytest

- 자동화 플로우 성공/실패 케이스
- 한글 토큰 처리 검증

### Playwright

- 튜토리얼 흐름 완료 여부
- 전자동 모드 실행 후 결과 확인

---

## 8) 기대 효과

- 도입 실패율 50% 감소
- 운영 인력 부담 40% 감소
