## 현재 기능 사용 경험

- **검색**: 특정 키워드 검색은 가능하나 보안 등급 필터 없음
- **지식 구조**: 라벨/관계는 있으나 기밀 등급 분리 미흡
- **AI 질의**: 답변 생성은 유용하지만 기밀 차단 정책 미흡

## 사용 불편점

| 불편점                       | 심각도 | 영향           |
| ---------------------------- | ------ | -------------- |
| 기밀 등급 기반 필터 부재     | 상     | 정보 유출 위험 |
| 접근 로그/감사 내보내기 부족 | 상     | 감사 대응 지연 |

## 변경 요구사항

1. 청크 **기밀 등급(RBAC)** 적용
2. 열람/수정/다운로드 **감사 로그 강화**
3. AI 질의 **민감 정보 차단** 및 출처 표시
