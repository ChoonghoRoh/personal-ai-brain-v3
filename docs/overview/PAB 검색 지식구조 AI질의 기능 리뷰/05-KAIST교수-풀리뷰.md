## 1) 현재 기능 설명 및 사용 흐름

### 검색

- 논문/보고서 키워드·의미 검색으로 관련 연구 탐색

### 지식 구조

- 라벨/관계로 연구 주제 구조화 가능
- 사용자별 공간 분리가 없어 협업 충돌 위험

### AI 질의

- 초안 작성에 유용하나 인용/출처 관리가 미흡

---

## 2) 사용 후기

- **좋은 점**: 검색 품질과 지식 구조화 기능이 연구에 유용
- **아쉬운 점**: 인용/연동 체계가 약함

| 구분 | 문제                     | 영향                 |
| ---- | ------------------------ | -------------------- |

| 협업 | 학생별 워크스페이스 부재 | 수정 충돌            |
| 연동 | Jupyter/Notion 연동 부족 | 연구 파이프라인 단절 |

3. **API/플러그인 문서 및 예제 제공**

## 5) 업그레이드/변경/통합 시나리오

- **통합 시나리오 A**: 검색 결과 → 인용 자동 생성 → AI 질의 요약

## 6) QA/보안/성능 관점

- API 접근 감사 로그

## 7) 테스트 시나리오 제안

- 인용 포맷 변환 로직 테스트

- 학생/교수 권한 UI 차이 확인
