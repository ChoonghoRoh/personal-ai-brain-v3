## 현재 기능 사용 경험

- **청크 생성**: 창업팀별 사업계획서·피칭 덱·고객 인터뷰 노트, 주 단위 피벗.  
  **느낀 점**: "아이디어를 주 3번 바꾼다. 삭제·재생성 말고 버전 쌓아두고 비교하고 싶다."
- **청크 승인**: 멘토 검토 후 승인, 거절 시 피드백 메모.  
  **느낀 점**: "Slack·Notion에서 피드백 주는데, 청크 화면에서 @mention·메시지 남기면 협업이 빨라진다."
- **청크 관리**: 팀별 청크 폴더처럼 분리, 투자자에게 공유할 청크만 링크.  
  **느낀 점**: "공유 링크 없음. 특정 청크 5개만 보여주려면 복사·붙이기 해야 한다."

---

## 사용 불편점

| 불편점 | 심각도 | 영향 |
|--------|--------|------|
| 버전 관리 부재 | 상 | 피벗 전후 비교 불가 |
| 멘토 협업 도구 미흡 | 중 | Slack·Notion으로 분산 |
| 청크 공유 기능 없음 | 중 | 투자자에게 보여주기 어려움 |
| 빠른 실험 지원 부족 | 중 | 매번 전체 프로세스 반복 |

---

## 변경 요구사항

1. 버전 관리·Diff (수정 시 이전 버전 저장, V1 vs V3 비교 화면)  
2. 협업 도구 통합 (Slack·Discord·Notion 청크 링크 공유, 승인 시 @mention)  
3. 청크 공유 링크 (선택 5개 → 공유 링크 생성, 유효기간 7일·30일·무제한)  
4. 빠른 실험 모드 (임시 청크·승인 없이 테스트 후 나중에 승인/삭제)

---

## 업그레이드 시나리오 제안

- **옵션 A**: 창업팀별 워크스페이스 → 멘토 검토(Slack 연동) → 투자자 공유(공유 링크)  
- **옵션 B**: Git 스타일 브랜치(피벗 전 스냅샷), 버전 태그 (#v1-초기아이디어)

---

## 기대 효과 (창업 생태계 관점)

| 개선 항목 | 현재 문제 | 개선 후 |
|-----------|-----------|---------|
| 피벗 추적 | 이전 아이디어 손실 | 버전 관리 → 학습 곡선 가시화 |
| 멘토 협업 | 도구 분산 | 통합 플랫폼 → 소통 50% 단축 |
| 투자 유치 | 자료 준비 2~3일 | 공유 링크 → 즉시 제공 |

아이디어 검증 2개월→3주, 투자자-창업팀 매칭 효율·대학 창업 성과 가시화.
