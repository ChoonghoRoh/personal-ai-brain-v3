## 현재 기능 사용 경험

- **청크 생성**: 월 1회 정부 과제 50페이지, 연구원 2명 분담 작성 후 통합.  
  **느낀 점**: "IT 전문가 없음. 2단계에서 병합 버튼 못 찾아 30분 헤맴. 동영상 튜토리얼 있으면 좋겠다."
- **청크 승인**: 조이사 단독 승인, AI 추천은 이해 못 해 무시.  
  **느낀 점**: "AI 추천이 영어로 나와 연구원들이 이해 못 한다. 한글로만 설정 가능해야 한다."
- **청크 관리**: 거의 미사용, 승인 후 끝. 검색 시 라벨 없어 찾기 어려움.  
  **느낀 점**: "라벨을 왜 붙이는지 모르겠다. 시스템이 자동으로 붙여주면 안 될까. 인력 부족해 관리 시간 없다."

## 사용 불편점

| 불편점 | 심각도 | 영향 |
|--------|--------|------|
| 사용법 학습 어려움 | 상 | IT 비전문가 혼자 배우기 힘듦 |
| 영문 UI/라벨 | 중 | 연구원 이해 못 함 |
| 자동화 부족 | 상 | 모든 단계 수동 |
| 유지보수 부담 | 중 | 서버 설정·업데이트 전문 지식 필요 |

---

## 변경 요구사항

1. 초보자 모드 (단계별 Tooltip, "다음 할 일" 가이드 화살표)  
2. 완전 한글화 (UI·라벨·오류 메시지 한글, 영문은 설정 선택 시만)  
3. 완전 자동화 옵션 ("전자동 모드": 업로드 → 분할·승인·라벨, 결과만 확인)  
4. 관리형 서비스(SaaS) (자체 서버 대신 클라우드, 월 구독·유지보수 없음)

---

## 업그레이드 시나리오 제안

- **옵션 A**: 파일 업로드 → 가이드 모드(단계별 화살표) → 전자동 승인·라벨(한글) → 완료 알림  
- **옵션 B**: cloud.p-ai-brain.com, 로그인 후 즉시 사용, 자동 백업·업데이트
