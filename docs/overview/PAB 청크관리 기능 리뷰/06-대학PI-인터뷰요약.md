## 현재 기능 사용 경험

- **청크 생성**: 학기별 1~2회 강의·과제 보고서, 텍스트 위주.  
  **느낀 점**: "UI가 복잡해 학생들이 어려워해 제가 대신 업로드. '파일 업로드 → 자동 처리' 버튼 하나면 좋겠다."
- **청크 승인**: 거의 전체 승인만, AI 추천 영문이라 학생들이 무시.  
  **느낀 점**: "한글 전용 모드 있으면 좋겠다."
- **청크 관리**: 학생이 과거 실험 찾을 때만, 라벨 추가는 거의 안 함.  
  **느낀 점**: "라벨을 시스템이 자동으로 붙여주면 안 될까. 인력 부족해 관리 시간 없다."

---

## 사용 불편점

| 불편점 | 심각도 | 영향 |
|--------|--------|------|
| UI 복잡도 | 중 | 사용법 학습 2시간 |
| 한글 지원 부족 | 중 | 영문 라벨 → 학생 이해 못 함 |
| 자동화 부족 | 중 | 모든 단계 수동 |
| 예산 제약 미고려 | 하 | 대용량 위주 설계 |

---

## 변경 요구사항

1. 간단 모드: "파일 업로드 → 자동 분할 → 자동 승인 → 자동 라벨", 전문가 모드 전환 가능  
2. 한글 우선 옵션 (AI 추천 한글 우선, 영문 괄호 병기)  
3. 자동 라벨링 강화 (승인 시 자동 적용, 신뢰도 0.7 이상만 등)  
4. 경량화 옵션 (텍스트 전용, 10명 미만·1GB 이하 프리셋)

---

## 업그레이드 시나리오 제안

- **옵션 A**: 파일 업로드 → 자동 처리 버튼 → AI 최적 분할·자동 승인·자동 라벨(한글) → 완료 알림  
- **옵션 B**: "대학 강의 자료"·"학생 과제" 템플릿 (강의명·주차·학생명·제출일 자동 태깅)
