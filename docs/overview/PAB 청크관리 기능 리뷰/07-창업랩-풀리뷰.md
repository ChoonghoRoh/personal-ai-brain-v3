### 4.7 창업랩 디렉터 (송창업 대표)

**조직**: 대학 창업지원단 창업랩 (예비 창업팀 10개 입주)  
**경력**: 5년 (스타트업 엑셀러레이터)  
**팀 규모**: 입주팀 10개 (각 3~5명)  
**주요 업무**: 창업팀 멘토링, 투자 유치 지원, 아이디어 검증

#### 4.7.1~4.7.5 요약

- **사용 경험**: 창업팀별 사업계획서·피칭 덱·고객 인터뷰 노트, 주 단위 피벗. 버전 쌓아두고 비교 희망. 멘토가 Slack·Notion에서 피드백→청크 화면에서 @mention·메시지 희망. 공유 링크 없어 투자자에게 복사·붙이기.
- **불편점**: 버전 관리 부재, 멘토 협업 도구 미흡, 청크 공유 기능 없음, 빠른 실험 지원 부족.
- **요구사항**: 버전 관리·Diff, Slack·Discord·Notion 연동·@mention, 청크 공유 링크(유효기간), 임시 청크(승인 없이 테스트).
- **시나리오**: 창업팀별 워크스페이스→멘토 검토(Slack 연동)→투자자 공유(공유 링크); 또는 Git 스타일 브랜치·버전 태그.
- **기대 효과**: 피벗 추적 버전 관리, 멘토 협업 50% 단축, 투자 유치 공유 링크 즉시. 아이디어 검증 2개월→3주.

**풀 리뷰 원문**: 원본 문서 §4.7 참조.
